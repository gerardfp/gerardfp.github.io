Mètode constructor

Una classe conté mètodes constructors. Un mètode constructor s'invoca en el moment en què es creen objectes d'aquesta classe (amb la paraula reservada new).

La declaració d'un mètode constructor es similar a la dels mètodes, excepte que el nom del constructor és el mateix que el de la classe i no s'ha d'especificar el tipus de retorn.

class Alumne { String nom; Alumne() { // codi del constructor } } public class Main { public static void main(String[] args) { Alumne unAlumne = new Alumne(); } }
