Ejercicio 1: Persona

Crea una clase llamada Persona

  • Añade a la clase tres campos de tipo String: nombre , apellido1 y apellido2

  • Añade un constructor para inicializar los tres campos a partir de los parámetros.

  • Añade tres métodos setter/getter para obtener/establecer los campos.

  • Añade un método que retorne unString con el nombre completo con el formato: apellido1 apellido2, nombre

  • Añade un método que reciba un parámetro String y retorne un boolean indicando si algun campo es igual al parámetro pasado.

Crea un programa que utilice la clase Persona

Ejercicio 2: TextBox

Crea una clase llamada TextBox

  • Añade a la clase dos campos de tipo int: ancho y alto. Y otro de tipo String: texto

  • Añade un constructor que reciba un parámetro String para inicializar el campo texto.

  • Añade otro constructor que reciba un String, y dos int para incializar los campos ancho y alto.

  • Añade otro constructor que reciba dos int para incializar los campos ancho y alto.

  • Añade tres métodos setter para establecer texto, ancho y alto

  • Añade otro campo setter para establecer el texto, que además reciba un parámetro boolean. Si el parámetro es true y el texto no está vacío, el campo ancho se debe establecer a la longitud del campo texto y el campo alto a 1.

  • Añade el siguiente método toString:

    @Override public String toString() { return ancho + "x" + alto + "\n" + "┏" + "━".repeat(ancho) + "┓\n" + ("┃" + " ".repeat(ancho) + "┃\n").repeat((alto - 1) / 2) + (alto > 0 ? "┃" + " ".repeat((ancho - texto.length() + 1) / 2) + texto + " ".repeat((ancho - texto.length()) / 2) + "┃\n" : "") + ("┃" + " ".repeat(ancho) + "┃\n").repeat(alto / 2) + "┗" + "━".repeat(ancho) + "┛\n"; }

  • IMPORTANTE: todos los métodos anteriores deben asegurar que:

    • Los campos ancho y alto siempre son mayores que 0.

    • El campo ancho siempre es mayor o igual a la longitud del campo texto

    • El campo alto es mayor o igual 1 si el campo texto no está vacío.

    • El campo texto nunca debe ser null.

Crea un programa que utilice la clase TextBox

Ejercicio 3: List String Utils

Crea una clase llamada StringListUtils. Añade los siguientes métodos static:

  • method1: Cuenta el número de veces que aparece un texto en una lista.

    • Parametros:

      • List<String> param1 Una lista de textos

      • String param2 El texto a buscar

    • Retorno:

      • int El número de veces que aparece el param2 en el param1

  • method2: Junta los textos de una lista en un único texto, utilizando un separador.

    • Parametros:

      • List<String> param1 Una lista de textos

      • String param2 El separador

    • Retorno:

      • String Los textos del param1 unidos usando el separador param1

  • method3: Crea una lista uniendo dos listas. No modifica ninguna de las dos listas.

    • Parametros:

      • List<String> param1 Una lista de textos

      • List<String> param2 Otra lista de textos

    • Retorno:

      • List<String> Una lista con los textos de param1 y los de param2

  • method4: Crea una lista quitando elementos de una lista que coinciden con un texto. No modifica la lista original.

    • Parametros:

      • List<String> param1 Una lista de textos

      • String param2 El texto a quitar de la lista

    • Retorno:

      • List<String> Una lista con los textos de param1 que no son iguales a param2

  • method5: Crea una lista quitando elementos de una lista que estan en otra lista.. No modifica las listas originales.

    • Parametros:

      • List<String> param1 Una lista de textos

      • List<String> param2 Una lista con los textos a quitar

    • Retorno:

      • List<String> Una lista con los textos de param1 que no están en param2

  • method6: Crea una lista con un texto repetido un número de veces.

    • Parametros:

      • String param1 El texto a repetir

      • int param2 El número de veces que se debe repetir/p>

    • Retorno:

      • List<String> Una lista con el param1 repetido param2 veces.
