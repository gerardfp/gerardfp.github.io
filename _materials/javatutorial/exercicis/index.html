Ejercicio 5: Object Racing

Programa un simulador de carreras de coches rudimentario.

El programa ha de tener las siguientes clases:

--- title: Object Racing --- classDiagram Carrera o-- Coche class Coche { +String piloto +int distancia +int velocidad +static int maxVelocidad +Coche(String piloto) +acelerar(int velocidad): void +mover(): void } class Carrera { +List~Coche~ coches +int distancia +Carrera(int distancia) +iniciar(): void +inscribir(Coche coche): void +primeraPosicion(): Coche }

La implementación de los métodos debe seguir las siguiente especificaciones:

Coche

  • Coche(String piloto):

    Asigna al campo piloto el valor pasado por parámetro.

  • void acelerar(int velocidad):

    Suma a la velocidad del coche el valor pasado por parámetro.

    Se debe asegurar que la velocidad no queda inferior a 1 ni superior a maxVelocidad.

  • void mover():

    Suma al campo distancia el campo velocidad.

Carrera

  • Carrera(int distancia):

    Asigna al campo distancia el valor pasado por parámetro.

    Además, establece la maxVelocidad de los coches a un valor calculado: distancia/15

  • Coche primeraPosicion():

    Devuelve un Coche que tenga el máximo valor en el campo distancia.

  • void inscribir(Coche coche):

    Añade el Coche pasado por parámetro a la lista coches.

  • El método void iniciar():

    void iniciar(){ while(true) { System.out.println("-".repeat(distancia)); coches.forEach(coche -> { coche.acelerar(Coche.maxVelocidad /2-random.nextInt(Coche.maxVelocidad)); coche.mover(); System.out.println(" ".repeat(coche.distancia) + coche.piloto + " (" + coche.velocidad + ")"); }); Coche primero = primeraPosicion(); if (primero.distancia >= distancia) { System.out.println("Ha ganado " + primero.piloto); return; } try { Thread.sleep(1000); } catch (InterruptedException e) {} } }
