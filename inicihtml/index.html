Desenvolupament de la pràctica

La pràctica consisteix en crear una pàgina web.

Aquesta web s'haurà de publicar a GitHub Pages amb el nom smx-m8-uf5-practica2 S'haurà de preparar un servidor web local, per al desenvolupament de la web. El desenvolupament de la web haurà de complir, al menys, els següents criteris:
  • La pàgina ha de seguir la recomanació indicada a "Anatomia d'un document HTML"
  • Ha de contenir els següents elements HTML:
    • Una capçalera principal
    • Dues capçaleres secundàries
    • Paràgrafs
    • Paraules destacadaes
    • Enllaços que portin a altres pàgines d'internet
    • Imatges
    • Els caracters especials < > " ' &
    • Comentaris
  • A més a més s'hauràn d'especificar les següents metadades
    • Títol
    • Codificació de caracters
    • Autor
    • Descripció
    • Favicon
    • Idioma principal del document
  • Per últim, s'hauran d'enllaçar dos arxius CSS i Javascript:
    • style.css
    • script.js

    El contingut dels arxius ha de ser aquest:
