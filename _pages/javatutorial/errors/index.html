Errors comuns

  • Posar punt-i-coma després de if, for, while:

    if (tal == cual); { } for (int i=0; i < 10; i++); { }

    Excepte en comptades excepcions, no es posa punt-i-coma.

  • Posar un igual = en la condició de if, for, while:

    if (a = b){ }

    Quan es vol fer una comparació s'han d'utilitzar dos iguals ==.

  • Comparar Strings amb ==:

    if (variable == "hola"){ }

    Per a comparar Strings és amb .equals():

    if (variable.equals("hola")){ }
